# EdgeSpark Gemini CLI Extension

Official Gemini CLI extension for building EdgeSpark apps.

## What's Included

- **building-edgespark-apps** skill — Development workflows, server patterns, web patterns, auth config, and deployment guidance for EdgeSpark projects

## Installation

```bash
gemini extensions install https://github.com/edgesparkhq/gemini-extensions
```

## When It Activates

The skill activates when a project has `edgespark.toml`, the user mentions EdgeSpark, or work involves the EdgeSpark CLI, server SDK, storage/auth/database workflows, deployment, or `@edgespark/web`.
